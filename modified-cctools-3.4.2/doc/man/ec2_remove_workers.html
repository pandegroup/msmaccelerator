ec2_remove_workers(1)

NAME

ec2_remove_workers - remove work_queue_worker instances on the Amazon EC2 service.

SYNOPSIS

ec2_remove_workers [options] <num-workers>

DESCRIPTION

ec2_remove_workers removes work_queue_worker(1) running on the Amazon EC2 service and deletes the EC2 instances running them. It calls EC2kill that is part of the EC2 API tools to delete the EC2 instances.

OPTIONS

-a Remove all running workers and terminate their EC2 instances.
-i <image_id>
EC2 OS image ID of instances running the workers. Default = ami-fa01f193.
-h Show help message.

EXIT STATUS

On success, returns zero. On failure, returns non-zero.

EXAMPLES

Remove 10 work_queue_worker instances and delete the EC2 instances running them. 
ec2_remove_workers 10
