parrot_mkalloc(1)

NAME

parrot_mkalloc - create a directory with a space allocation

SYNOPSIS

parrot_mkalloc <path> <size>

DESCRIPTION

parrot_mkalloc creates a new directory with a space allocation. As the name suggests, the command only runs correctly inside of the Parrot virtual file system, on file servers where space allocation is enabled.

The path argument gives the new directory to create, and the size argument indicates how large the space allocation should be. The latter may use metric units such as K, M, B, etc to indicate kilobytes, megabytes, gigabytes, and so forth.

OPTIONS

-h Show help text.

EXIT STATUS

On success, returns zero. On failure, returns non-zero. If the command is attempted on a filesystem that does not support space allocation, it will give the following error: 
parrot_mkalloc: This filesystem does not support allocations.

EXAMPLES

To create a space allocation of ten gigabytes on a Chirp server: 
% parrot_run bash
% cd /chirp/myserver.somewhere.edu
% parrot_mkalloc bigdir 10G
% exit
